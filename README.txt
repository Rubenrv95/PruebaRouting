INSTRUCCIONES DE INSTALACIÓN

-------------------------------------


1. Mediante la consola de comandos, ingrese el siguiente comandos:

"git clone https://github.com/Rubenrv95/PruebaRouting"


2. Ahora, diríjase a la carpeta del proyecto ingresando el comando:

"cd PruebaRouting"


3. Ingrese: "npm install"


4. Ingrese: "npm start"


5. Usando la app móvil Expo, escanee el código QR que aparece en pantalla.


6. Espere a que compile el programa y listo. Puede ingresar al sistema con la siguiente cuenta:

ruben@correo.cl
contraseña: 123456


Notas:
Asegurese de tener instalados Node.js y npm en sus últimas versiones
